# ⚡ Three-Phase Squirrel-Cage Induction Motor Design Software

A comprehensive Windows-based engineering application for the preliminary design, geometric calculation, and analysis of three-phase squirrel-cage induction machines. 

This tool was developed to streamline the electrical machine design process, taking user-defined constraints and automatically performing the complex electromagnetic and physical calculations required to size a motor.

## ✨ Key Features

*   **Customizable Machine Specifications:** Input desired target parameters including Voltage, Power, Frequency, and Number of Poles.
*   **Targeted Design Objectives:** Optimize the machine's design based on specific goals: **Cost-Effective**, **High Efficiency**, or a **Balanced** design.
*   **Step-by-Step Workflow:** 
    *   Executes the Stator design first.
    *   Provides the user the flexibility to review, modify, or retain default Rotor parameters before finalizing the Rotor design.
*   **Material & Configuration Selection:** Choose phase connections, winding types, and magnetic lamination materials.
*   **Interactive 2D Visualization:** Automatically generates a 2D cross-sectional view of the designed machine, visually displaying the different calculated physical lengths and dimensions.
*   **Complete Mathematical Transparency:** Displays every calculated quantity alongside the exact engineering formula used to derive it, ensuring the design process is not a "black box."
*   **Productivity Tools:** 
    *   Built-in **Presets** for quick testing and demonstration.
    *   **Save/Load Project** functionality to pause and resume designs.
    *   **Export to PDF** to automatically generate a formal engineering report of the designed machine.
